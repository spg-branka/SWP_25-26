# Allergene in typischen Speisen

## Legende (Allergene)

- Gluten (Getreide)
- Milch
- Ei
- Soja
- Nüsse (Schalenfrüchte)
- Erdnuss
- Sellerie
- Senf
- Sesam
- Fisch
- Krebstiere/Weichtiere
- Sulfite
- Lupinen

## Mapping: Speise -> häufig relevante Allergene

- Schnitzel mit Pommes -> Gluten, Ei; ggf. Milch; ggf. Sellerie/Senf (Saucen)
- Chicken Nuggets -> Gluten, Ei; ggf. Milch
- Burger (Beef/Chicken/Veggie) -> Gluten; ggf. Milch (Käse/Sauce); ggf. Ei (Mayo); ggf. Senf; ggf. Sesam (Bun);
  bei Veggie oft Soja; ggf. Sellerie (Saucen/Gewürz)
- Cheeseburger -> Gluten, Milch; oft zusätzlich Ei, Senf, Sesam; ggf. Soja/Sellerie
- Veggie-/Vegan-Burger -> Gluten (je nach Bun/Patty), oft Soja; ggf. Lupinen; ggf. Sesam; ggf. Senf;
  bei vegetarisch ggf. Milch/Ei
- Pizza -> Gluten, Milch; ggf. Soja (verarbeiteter Belag)
- Pasta (Bolognese/Napoli) -> Gluten; ggf. Ei (bei Eiernudeln); ggf. Sellerie (Sauce)
- Lasagne -> Gluten, Milch; häufig Ei; ggf. Sellerie
- Käsespätzle/Spätzle -> Gluten, Ei, Milch
- Palatschinken/Pfannkuchen -> Gluten, Ei, Milch; ggf. Nüsse (Füllung/Topping)
- Cremesuppe -> Milch; häufig Sellerie; ggf. Gluten (Einbrenn)
- Gulasch/Ragout/Schmorgerichte -> häufig Sellerie; ggf. Gluten (Bindung); ggf. Senf
- Salat + Dressing -> ggf. Milch (Joghurt/Parmesan); ggf. Ei (Caesar/Mayo); ggf. Senf (Vinaigrette);
  ggf. Nüsse/Sesam (Toppings)
- Sandwich/Toast/Weckerl -> Gluten; häufig Milch (Käse); ggf. Ei (Aufstriche); ggf. Senf; ggf. Sesam (Gebäck)
- Müsli/Granola + Joghurt -> Milch; ggf. Gluten; häufig Nüsse; ggf. Sulfite (Trockenfrüchte)
